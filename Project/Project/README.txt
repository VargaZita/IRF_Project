﻿Projekt: Az árukeresőhöz hasonló termékösszehasonlító alkalmazás. 
Termékek: Mobiltelefonok

Tervezett funkciók:
Adatvizualizáció: Termékek értékelésének megjelenítése diagramon (csillagok száma) + értékelés hozzáadása
"Szerencsekerék": 
	A vásárlónak lehetősége van kedvezményt sorsolni magának, amihez kap egy kódot,
	amit az eszköz megvásárlásakor az értékesítő weboldalán a megfelelő helyre beírva
	érvényesíthet.
