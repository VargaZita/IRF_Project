﻿Projekt: Az árukeresőhöz hasonló termékösszehasonlító alkalmazás. 
Termékek: Mobiltelefonok

Tervezett funkciók:
Adatvizualizáció: Termékek értékelésének megjelenítése diagramon (csillagok száma) + értékelés hozzáadása
"Szerencsekerék": 
	A vásárlónak lehetősége van kedvezményt sorsolni magának, amihez kap egy kódot,
	amit az eszköz megvásárlásakor az értékesítő weboldalán a megfelelő helyre beírva
	érvényesíthet.

A videó linkje: https://drive.google.com/file/d/1W_3yYELtwCT_s9XebJZTEM-n_uJ-J5xF/view?usp=sharing

Feladatok:
1. B - Adatbázis létrehozása, kapcsolódás és legalább egy tábla adatainak beolvasása
2. B - Kifejtett setter ág használata, értékadásra értelemszerű funkciók hívása
3. C - Diagram rajzolás + kapcsolódó adatok megjelenítése (pl.: DataGridView)
4. A - Véletlenszám generálás és felhasználás
